# Personal Budget & Expense Tracker

## Project Overview

The Personal Budget & Expense Tracker is a simple web application built using HTML and CSS. It allows users to record expenses, organize them into categories, and view them in a structured table. This project serves as the foundation for future enhancements where JavaScript will be added to make the tracker fully interactive.

---

## Features

- Responsive webpage layout
- Expense input form
- Category dropdown with five expense categories
- Expense history table with sample data
- Budget tracker logo
- Embedded budgeting tips YouTube video
- Collapsible "How to Use This Tracker" section
- Styled table with alternating row colors
- Hover effects for table rows
- Focus effects on form inputs
- Advanced CSS selectors for improved styling

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

No installation or additional software is required.
